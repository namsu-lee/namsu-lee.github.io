## 옵시디언 마크 다운 문법

잘 모르면 편집기 우클릭 하면 나옵니다.

### 제목

```markdown
# 뒤에 띄어쓰기를 해야 제목으로 인식

# 제목 1

## 제목 2
```

### 글씨

```markdown
** 굵은 글씨 **
~~ 취소선 ~~
== 강조 ==

> 인용문. 개행 시 자동으로 >가 입력됨
```

### 리스트

```markdown
- 일반 리스트
- [ ] 태스크 리스트
- [x] 태스크 리스트

1. 숫자 리스트
```

### 링크

```markdown
[[파일명]] 파일 링크
[[파일명.md^위치id]] 파일의 위치 링크
![[파일명.md#위치id]] 파일의 위치의 내용을 가져옴
![[파일명.canvas]] 캔버스의 내용을 가져옴
![[이미지명]] 이미지 링크
![[이미지명|가로크기]] 이미지 링크 크기 조정
[링크명](외부링크) 외부 링크(http:// 혹은 https:// 가 있어야 작동)
```

### 강조

강조는 모두 여러 줄 입력 가능

```
> [!note] 노트
> [!abstract] 요약
> [!info]
> [!warning]
> [!danger]
> [!example]
> [!todo] 할 일
> [!tip]
> [!quote] 인용
> [!note]+ 접을 수 있는 강조문. 편 상태
> [!note]- 접을 수 있는 강조문. 접은 상태
```

## 단축키

| 기능           | 단축키      |
| -------------- | ----------- |
| 줄 복사        | Control + C |
| 줄 잘라내기    | Control + X |
| 명령어 창 열기 | Control + P |
| 속성 생성      | Control + ; |

## 템플릿

템플릿 파일에서 원본 모드(Source mode)를 사용하면 변수 사용 가능

| 변수      | 설명                                     |
| --------- | ---------------------------------------- |
| {{title}} | 제목                                     |
| {{date}}  | 오늘의 날짜입니다. 기본 형식: YYYY-MM-DD |
| {{time}}  | 현재 시간입니다. 기본 형식: HH:mm        |

## 동기화 방법

윈도우는 원드라이브 사용하면 간단히 해결됩니다.  
개인적으로 발생한 문제는 Github Pages를 이용해 블로그를 만드려고 심볼릭 링크를 사용했는데 원드라이브는 심볼릭 링크를 허용하지 않습니다.  
그래서 원드라이브는 설정을 저장하는 곳으로만 사용하는 중입니다.

## 코어 플러그인

일단 내가 쓰는 것들만 정리

| 한국어 이름         | 영어 이름                 | 기능                                         |
| ------------------- | ------------------------- | -------------------------------------------- |
| 백링크, 나가는 링크 | Backlinks, Outgoing links | 링크된 노트를 보여줌                         |
| 캔버스              | Canvas                    | 노트 연결을 시각화                           |
| 명령어 팔레트       | Command palette           | 명령어 창 활성화(Control + P)                |
| 파일 복원           | File recovery             | 자동 저장 및 스냅샷 보관                     |
| 파일 탐색기         | Files                     | 프로그램 내부 파일 브라우저 활성화           |
| 그래프 뷰           | Graph view                | 노트 간 연결 시각화                          |
| 개요                | Outline                   | 노트의 제목을 개요 형태로 사이드 바에 보여줌 |
| 빠른 전환기         | Quick switrcher           | Control + O를 눌러 노트 검색                 |
| 검색                | Search                    | 검색(끄면 단축키도 안 먹힘)                  |
| 슬래시 명령어       | Slash commands            | /를 이용해 명령 사용                         |
| 노트 작성기         | Note composer             | 노트 추출 및 병합                            |
